# The Eyes Have It
## An Exercise Styling Eyes

MIT xPRO - Week 8

This project was part of the course work required in the MIT xPRO Full-Stack development course.

Description:

An exercise to track mouse movement by cloning two eyes that will follow the mouse around the screen.

Versioning: 1.1.1

Coming Improvements: 
Independent Movement with each eye
Selectable Eye Color
